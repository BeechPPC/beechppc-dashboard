# Google Ads Sitelink Best Practices

## Character Limits

**Critical constraints that MUST be followed:**
- **Link Text**: Maximum 25 characters including spaces
- **Description Line 1**: Maximum 35 characters including spaces
- **Description Line 2**: Maximum 35 characters including spaces

Google Ads will reject sitelinks that exceed these limits.

## Link Text Guidelines

### Writing Principles
1. **Action-oriented**: Use verbs that encourage clicks (Entdecken, Jetzt kaufen, Mehr erfahren)
2. **Specific**: Reference the actual page content, not generic terms
3. **Benefit-focused**: Highlight what the user will gain
4. **Brand-appropriate**: Match the tone and language of the landing page

### Common Patterns
- Product categories: "[Product] entdecken" (e.g., "Seren entdecken")
- Benefit-driven: "[Benefit] erzielen" (e.g., "Strahlende Haut jetzt")
- Action calls: "Jetzt [action]" (e.g., "Jetzt shoppen")
- Direct naming: "[Category]" when clear (e.g., "Gesichtspflege")

### Avoid
- Generic phrases like "Klick hier" or "Mehr Info"
- Excessive punctuation or special characters
- ALL CAPS (except for acronyms)
- Trademark symbols unless absolutely necessary
- Keyword stuffing

## Description Lines Guidelines

### Writing Principles
1. **Complementary**: Lines should work together to tell a cohesive story
2. **Clear value**: Each line should communicate a distinct benefit or feature
3. **Natural breaks**: Split at logical phrase boundaries
4. **Keyword-aware**: Include relevant search terms naturally

### Structure Patterns

**Pattern 1: Feature + Benefit**
- Line 1: "Hochkonzentrierte Wirkstoffe"
- Line 2: "Für sichtbare Resultate"

**Pattern 2: Action + Outcome**
- Line 1: "Tiefenwirksame Seren entdecken"
- Line 2: "Für strahlende und gepflegte Haut"

**Pattern 3: Benefit + Benefit**
- Line 1: "Intensive Feuchtigkeit & Pflege"
- Line 2: "Für alle Hauttypen geeignet"

**Pattern 4: Product + USP**
- Line 1: "Exklusive Seren bei CAREISTA"
- Line 2: "Sorgfältig ausgewählte Marken"

### Avoid
- Repeating information from Link Text
- Incomplete sentences that confuse users
- Overly promotional language ("Beste", "Günstigste")
- False urgency ("Nur heute", "Letzte Chance")
- Breaking mid-word or mid-compound

## German Language Considerations

### Compound Words
German compounds can be long. Break descriptions before or after compounds, not within them:
- ✓ "Gesichtsserum für jeden Hauttyp"
- ✗ "Gesichts-|serum für jeden Hauttyp"

### Formal vs. Informal
Match the landing page's tone:
- Formal (Sie): "Entdecken Sie", "Für Ihre Haut"
- Informal (du): "Entdecke", "Für deine Haut"

Most modern e-commerce uses informal tone (du).

## Content Extraction Strategy

### Priority Order for Content Mining
1. **Title Tag**: Often contains the main value proposition
2. **Meta Description**: Usually well-optimized marketing copy
3. **H1**: The primary page heading, often concise and clear
4. **Category Text**: Detailed content explaining benefits and features

### Extraction Techniques
- Identify key benefits and features from category text
- Extract action verbs and compelling phrases
- Note unique selling propositions (USPs)
- Identify target audience language and tone
- Find natural phrase boundaries for description line splits

## Quality Checklist

Before finalizing sitelink text, verify:
- [ ] Link Text ≤ 25 characters
- [ ] Description Line 1 ≤ 35 characters
- [ ] Description Line 2 ≤ 35 characters
- [ ] No repetition between Link Text and Descriptions
- [ ] Action-oriented and benefit-focused language
- [ ] Grammatically correct and natural German
- [ ] Tone matches landing page content
- [ ] No Google Ads policy violations
- [ ] Lines split at natural phrase boundaries
